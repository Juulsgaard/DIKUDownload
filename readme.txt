For at bruge dette script skal du have "PHP cli" installeret.

Inden at du kan bruge scriptet skal du logge ind på Absalon i en browser.
Efter at have gjort dette skal du kopiere indholdet af den cookie ved navn "canvas_session" og indsætte den i config.json filen.

Derefter kan du vælge hvor filerne skal placeres på din computer via "save_location" configurationen.

Konfigurationen "unix" skal være true hvis du bruger Linux eller OSX. Den skal være false hvis du bruger Windows

Når dette er på plads skal du bare køre scriptet i cmd line

> php -f downloadDIKU.php
